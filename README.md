# Fun with Math

A playful, kid-friendly arithmetic game designed to make learning math fun and engaging.

## Live App
https://fun-with-math-rouge.vercel.app/

## Features
- Customizable game duration (1–5 minutes)
- Practice addition, subtraction, multiplication, and division
- Select difficulty (1-digit, 2-digit, 3-digit)
- Reward-based system using candies instead of scores
- Pressure-free gameplay with no per-question timer

## Why this project
This project was built to explore how simple, interactive tools can make learning enjoyable for kids.

## Tech Stack
- HTML
- CSS
- JavaScript
- Deployed on Vercel
